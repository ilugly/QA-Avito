Тестирование API микросервиса объявлений

Описание
Автоматизированные тесты для проверки API микросервиса управления объявлениями.

Предварительные требования
Python 3.8+

Установка и запуск

1. Установите зависимости:
pip install requests pytest

2. Запустите тесты:

Как Python скрипт:
python test_api.py

Через pytest:
pytest test_api.py -v

Структура проекта

Задание 1:

- BUGS1.md - отчет о найденных багах

Задание 2.1:

- test_api.py - основные автоматизированные тесты

- TESTCASES.md - описание тест-кейсов

- BUGS2.1.md - отчет о найденных багах

- README.md - инструкция по запуску
